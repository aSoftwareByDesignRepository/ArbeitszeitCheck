# Screenshots Directory

This directory should contain screenshots for the Nextcloud App Store:
- screenshots/dashboard.png - Main dashboard view
- screenshots/time-tracking.png - Time tracking interface
- screenshots/compliance.png - Compliance dashboard
- screenshots/reports.png - Reports view

Screenshots should be:
- At least 1280x720 pixels
- PNG format
- Showing the app in action with realistic data
- Following Nextcloud design guidelines
